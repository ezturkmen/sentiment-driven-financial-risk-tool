# 📊 **Sentiment-Driven Financial Risk Assessment Tool**

## **Project Overview**

This tool bridges the gap between financial news and quantitative risk assessment. Developed with a focus on the **Dublin financial market**, it automates the process of fetching economic headlines, analyzing sentiment via AI, and generating executive-level reports.

## **Key Features**

* **NLP Analysis:** Utilizes FinBERT (Financial BERT) for high-accuracy sentiment detection.
